LurkBot

LurkBot is a Discord bot designed to track 4chan threads and provide updates. Below you'll find a list of available commands and their usage.

Commands

!track <thread_url>

Starts tracking a specified 4chan thread.

Usage:

!track https://boards.4chan.org/po/thread/12345678

!add_user @username

Adds a user to the DM list to receive updates.

Usage:

!add_user @username

!remove_user @username

Removes a user from the DM list.

Usage:

!remove_user @username

!add_channel #channelname

Adds a channel to the update list.

Usage:

!add_channel #channelname

!remove_channel #channelname

Removes a channel from the update list.

Usage:

!remove_channel #channelname

!bot_status

Checks the status of the bot.

Usage:

!bot_status

!repost_last

Reposts the last tracked post.

Usage:

!repost_last

!remove_thread

Stops tracking the current thread.

Usage:

!remove_thread

!set_user_setting @username <setting> <value>

Sets a user-specific setting.

Usage:

!set_user_setting @username post_updates false

User Settings

post_updates

true or false

When set to false, the user will only receive bump limit updates at 450 and 500 posts.

Example:

!set_user_setting @username post_updates false

Notifications

Yellow: Thread has reached 450 posts.

Red: Thread has reached 500 posts.

Green: Bot has restarted.

Error Handling

If the bot encounters an error, it will notify users and channels with a red error embed.
